## 装修设计尺寸清单

### 客餐厅
* 鞋柜:
    + 深度: 400mm
    + 矮鞋区: 130-180mm
    + 中高鞋区: 220-320mm
    + 高帮鞋压: 400-500mm
    + 底部换鞋区: 150-200mm
    + 多做放鞋区 !多做放鞋区 !一家人的鞋真的挺多的!
* 餐边柜
    + 深度: 350mm-500mm(根据电器大小、尺寸个人需求选择)
    + 台面商度: 850-1000mm

### 厨房
* 橱柜:
    + 地柜深度: 600mm(常规深度)
    + 台面深度: 800mm-1000mm;如做商低台，炒菜洗菜高差100mm
    + 台面厚度: 12-20mm
    + 岩板常用 12mm、石英石15/20mm
    + 吊柜深度: 300-350mm
    + 吊柜距台面高度: 常用700-150mm
